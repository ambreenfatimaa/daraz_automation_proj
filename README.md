# Daraz Automation (Playwright + POM)

## What
Automation tests for Daraz.pk using Playwright (Python) with Page Object Model.

## Requirements
- Python 3.8+
- Playwright

## Setup (run locally)
1. Create & activate venv:
   - Windows: `python -m venv venv` then `venv\Scripts\activate`
2. Install deps:
   ```bash
   pip install -r requirements.txt
   playwright install
